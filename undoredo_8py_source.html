41 class UR:
42  '''provides undo/redo facility. There is support for unlimited undo's
43  but no support for branching (multiple paths) so all values are lost
44  beyond the position where the new object is added. For example, if you
45  perform a set of transformations t1, t2 and t3 which correspond to
46  schedule s1, s2 and s3 and we were to unddo the last transformation
47  and apply a new transformation t4, we would end up storing t1,t2,t4
48  and a schedule s1,s2,s4 i.e. t3 and s3 would be deleted.
49
